# 📚 Slay The Syllabus
### AI-Powered Smart Study Assistant

## 📖 Project Overview

**Slay The Syllabus** is an AI-powered learning platform designed to help students convert long and complex study materials into short, structured, and easy-to-understand revision resources.

Students can upload **PDFs, lecture notes, videos, images, or voice recordings**, and the platform automatically generates **summaries, key concepts, flashcards, quizzes, and revision notes**.

The main goal of this project is to **reduce study time, simplify learning, and help students prepare more efficiently for exams using AI-powered tools.**

---

## 🚀 Features

- 📄 **PDF & Notes Summarizer** – Convert long notes into short summaries  
- 🎥 **Video to Notes** – Extract key concepts from lecture videos  
- 🧠 **Key Concept Generator** – Identify important points from study material  
- 🃏 **Flashcard Generator** – Create flashcards for quick revision  
- ❓ **AI Quiz Generator** – Generate practice questions automatically  
- 🗺️ **Mind Map Generator** – Visualize topics for better understanding  
- 🎤 **Voice Notes to Summary** – Convert voice lectures into notes  
- 🌍 **Multilingual Summaries** – Translate summaries into different languages  
- 🤖 **Doubt Solving Chatbot** – Ask questions and get AI explanations  
- 📅 **AI Study Planner** – Plan study schedules effectively  

---

## ⚙️ How It Works

1. User uploads study material (**PDF, video, notes, image, or voice note**).
2. The AI processes the content using **Natural Language Processing (NLP)**.
3. Important concepts and information are extracted.
4. The system generates structured outputs like:
   - Summaries  
   - Key Points  
   - Flashcards  
   - Quizzes  
   - Mind Maps  

These materials help students **revise faster and understand concepts more effectively.**
