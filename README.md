# 📚 Slay The Syllabus
### AI-Powered Study Assistant

## Overview
Slay The Syllabus is an AI-powered platform that helps students convert long lectures, PDFs, videos, and notes into short summaries, key points, flashcards, quizzes, and revision notes.

## Features
- AI PDF Summarizer
- Video to Notes
- Flashcard Generator
- AI Quiz Generator
- Mind Map Generator
- Voice Notes to Summary
- Doubt Solving Chatbot
- Study Planner

## Tech Stack
Frontend: HTML, CSS, JavaScript  
Backend: AI APIs  
Deployment: Replit
